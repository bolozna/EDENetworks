
EDENetworks network analyzer
----------------------------
EDENetworks is a genetic network analyzer with GUI. It allows to create distance based networks from genetic data and do basic networks analysis.

Requirements
------------
In order for EDENetworks to work, some 3rd party software needs to be installed:
* Himmeli 3.0.1 at the directory "himmeli_3.0.1"
* Python 2.7 (some other versions might also work)
* Matplotlib
* Numpy
* Scipy
